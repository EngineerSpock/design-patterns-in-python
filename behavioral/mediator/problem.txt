В системе может быть любое количество экземпляров класса Participant.
Каждый экземпляр имеет целочисленный атрибут value изначально инициализируемый нулём.

Экземпляр participant может сказать say(), выдав значение, транслируя его всем участникам.
В данной точке во времени каждый участник обязан увеличить своё собственное значение на транслированное значение.

Пример:

- Два участника начинают со значений 0 соответственно
- Участник 1 транслирует значение 3. Теперь у нас Participant 1 value = 0, Participant 2 value = 3
- Участник 2 транслирует значение 2. Теперь у нас Participant 1 value = 2, Participant 2 value = 3
