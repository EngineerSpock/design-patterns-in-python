​Дан класс Sentence, конструктор которого принимает строку (такую как "hello world").

Вам нужно реализовать API таким образом, чтобы индексатор возвращал приспособленца через которого можно переводить конкретные слова из строки в верхней регистр. Вот пример использования такого API:

sentence = Sentence('hello world')
sentence[1].capitalize = True
print(sentence) # writes "hello WORLD"
